## 🜁 AETHERION CONSOLIDATED™ — ASSETS INDEX  
### *Global Asset Registry for the AETHERION Ecosystem*
Location: `.github/assets/`

This index catalogs all global identity assets used across the  
AETHERION CONSOLIDATED™ organization. These assets provide  
cross‑repository continuity for branding, documentation, and  
public‑facing materials.

---

## 🖼️ 1. Visual Identity Assets

### **AETHERION_CONSOLIDATED-AVATAR.jpg**  
- Primary avatar for GitHub org profile  
- Used in public-facing contexts where JPEG is preferred  
- High‑contrast, optimized for small display sizes

### **AETHERION_CONSOLIDATED-AVATAR.png**  
- Transparent PNG version of the org avatar  
- Used for documentation, READMEs, and UI mockups  
- Preferred for high‑fidelity rendering

### **AETHERION_CONSOLIDATED-SPLASH-MOBILE.png**  
- Mobile‑optimized splash artwork  
- Used for vertical layouts, mobile previews, and narrow aspect ratios  
- Recommended for onboarding screens and mobile-first designs

### **AETHERION_CONSOLIDATED-SPLASH-WIDESCREEN.png**  
- Widescreen splash artwork  
- Used for desktop layouts, banners, and wide aspect ratios  
- Ideal for READMEs, landing pages, and presentation headers

## 🧭 3. Usage Guidelines

- Assets in this directory are **global** and may be referenced by any  
  repository within the AETHERION CONSOLIDATED™ organization.
- Use raw GitHub URLs for cross‑repo embedding.
- Do not rename assets without updating this index.
- Add new assets only if they serve org‑wide identity or documentation needs.
